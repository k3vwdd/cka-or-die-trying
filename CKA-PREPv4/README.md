# CKA-PREPv4

This set includes 17 fresh CKA-style labs focused only on:

- Cluster Architecture, Installation and Configuration
- Troubleshooting
- Services and Networking

Each `Question-X` folder contains:

- `Question.bash`
- `LabSetUp.bash`
- `SolutionNotes.bash`
- `validate.sh`

Assumptions:

- Generic kubeadm environment
- Node names follow exam-style patterns such as `controlplane` and `node01`
- `kubectl` is configured for cluster-admin access on the working node
